# Customer Segmentation using K-Means Clustering

## Objective

Segment mall customers into different groups based on their annual income and spending score to support targeted marketing strategies.

## Machine Learning Algorithm

- K-Means Clustering

## Workflow

- Data Exploration
- Feature Selection
- Feature Scaling
- Elbow Method
- K-Means Clustering
- Silhouette Score Evaluation
- Cluster Visualization
- Customer Profiling
- Business Recommendations

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Business Outcome

The project identifies five customer segments that can be used for personalized marketing, loyalty programs, and strategic decision-making.
